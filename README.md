# 个人在线简历

基于 Next.js 构建的现代化个人简历网站。

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **部署**: GitHub Pages
- **特性**: 
  - 暗黑极客风格
  - 打字机效果
  - 流星背景动画
  - 交互式终端
  - 响应式设计

## 快速开始

### 安装依赖

```bash
npm install
# 或
pnpm install
# 或
yarn install
```

### 开发模式

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看效果。

### 构建生产版本

```bash
npm run build
```
